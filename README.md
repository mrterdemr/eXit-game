# eXit

A text-based adventure game written in Python. You are trapped in a dungeon with a friend — your choices determine who makes it out.

Inspired by **Mr. Robot, Season 4 Episode 11**.

## Requirements

- Python 3.8+

## Run

```bash
python eXit-game/eXit-game.py
```

## How to Play

Type a command at each prompt and press Enter. Input is not case-sensitive.

**Commands:**

```
leave      stay       examine    interact
take       read       light      look
yes        no
```

You can also use the exact phrases from Mr. Robot S4E11:

```
move the barrel        sit down next to my friend
enter tunnel           light a match
read note              get on the boat
```

## Story

You wake up trapped in a dungeon with your friend. A barrel stands in the corner. There is a way out — but your friend is too weak to follow.

Every decision you make changes the outcome. There is no single right answer.

**Possible endings:**
- You escape and head to a new world
- You stay — the dungeon collapses around you both
- You abandon your friend
- You hesitate too long and the tunnel collapses
